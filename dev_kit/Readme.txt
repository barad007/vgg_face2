This directory contains two demo scripts:

"demo_proj.m": Generate tightly cropped faces by using loosely cropped region on VGGFace2.

"demo_ijbb.m": Generate tightly cropped faces on IJB-B test set.

The outputs can be fed into the models released in the website for testing.



References for implementation:

1. Please download and run the face detector by following:

MTCNN: https://github.com/kpzhang93/MTCNN_face_detection_alignment


2. Model running on Caffe implementation:

1) Caffe: https://github.com/BVLC/caffe

2) Squeeze-and-Excitation Networks: https://github.com/hujie-frank/SENet


Or model running on MatCovNet implementation:

1) MatConvNet: https://github.com/vlfeat/matconvnet

2) Squeeze-and-Excitation Networks: https://github.com/albanie/mcnSENets
